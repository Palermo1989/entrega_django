Sobre este proyecto

Este sitio web permite gestionar una clínica médica privada. Podés registrar doctores, pacientes y estudios, ver sus datos, modificarlos y eliminarlos. También permite registrarse como usuario, subir un avatar y acceder a funciones exclusivas. La lista de doctores está disponible para todos, incluso sin iniciar sesión.
